# zerotier

[![Build](https://github.com/gsmlg-ci/zerotier/actions/workflows/build.yml/badge.svg)](https://github.com/gsmlg-ci/zerotier/actions/workflows/build.yml)

Docker image published to:
- `docker.io/gsmlg/zerotier`
- `ghcr.io/gsmlg-dev/zerotier`

## Usage

```bash
# From Docker Hub
docker pull gsmlg/zerotier:latest

# From GitHub Container Registry
docker pull ghcr.io/gsmlg-dev/zerotier:latest
```

## Build

```bash
docker build -t gsmlg/zerotier .
```
